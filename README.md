# Estruturas de Dados: Fila e Pilha em Java

Projeto prático desenvolvido para a disciplina de Estrutura de Dados, implementando as estruturas de Fila (Queue) e Pilha (Stack) manualmente em Java, sem o uso de bibliotecas prontas.

---

## Sobre o Projeto

O projeto consiste em dois sistemas independentes executados via terminal:

- **Sistema de Fila** — simula uma fila de pedidos de restaurante, seguindo o princípio FIFO (First In, First Out)
- **Sistema de Pilha** — simula uma pilha de livros a serem lidos, seguindo o princípio LIFO (Last In, First Out)

---

## Funcionalidades

### Fila (Restaurante)
- Adicionar pedido na fila
- Finalizar pedido (remover o primeiro da fila)
- Mostrar todos os pedidos
- Ver o próximo pedido
- Verificar se a fila está vazia

### Pilha (Livros)
- Adicionar livro à pilha
- Finalizar leitura (remover o topo da pilha)
- Mostrar todos os livros na pilha
- Ver o próximo livro a ser lido
- Verificar se a pilha está vazia

---

## Estrutura do Projeto

```
├── No.java          # Classe do nó utilizado nas estruturas
├── Fila.java        # Implementação da estrutura de Fila
├── MainFila.java    # Menu e execução do sistema de fila
├── Pilha.java       # Implementação da estrutura de Pilha
└── MainPilha.java   # Menu e execução do sistema de pilha
```

---

## Como Executar

**Pré-requisito:** Java JDK instalado.

```bash
# Compilar todos os arquivos
javac *.java

# Executar o sistema de Fila
java MainFila

# Executar o sistema de Pilha
java MainPilha
```
